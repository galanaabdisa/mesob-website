# MESOB Agaro One-Stop Service Center Website

This website helps residents of Agaro check which government services are available at MESOB (One-Stop Service Center) without visiting in person.

## Features
- Three languages: Afaan Oromo, Amharic, English
- List of all 28 government organizations at MESOB
- Services available at MESOB for each organization
- Search functionality
- Mobile-friendly design
- Works offline after first load

## How to Use
1. Open the website on any device
2. Select your preferred language
3. Browse or search for organization
4. Click organization to see services at MESOB

## For MESOB Administration
To update services:
1. Edit the `index.html` file
2. Find the `organizations` array in the JavaScript code
3. Update services as needed
4. Re-upload to GitHub
